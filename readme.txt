== Description ==

This plugin allows you to display various statistics as a widget.

You can display the number of posts, number of pages, number of categories, number of comments, number of pingbacks, number of tags, number of users, number of blogroll links and the Akismet spam count!

You can also customize the colour to use for the widget - Which means you can blend it into your theme!



It's very easy to setup the plugin - Simply enable it, alter the settings and then enable the widget!

== Installation ==

1. Upload `jr-stats.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Stats Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets tab.
